Weather App Software Development Plan

Software Development Plan

Phase 0: Requirements and Specifications

The objective of this assignment is to create a weather app that has 3 API calls. The three calls are first to get the location of the computer through their IP address using a geolocator. Once the latitude and longitude is obtained, that information is used to get the current weather for that specific location with the second API call. Thirdly, with the coordinates again, the same site is queried to obtain a 5-day, 3 hour forecast which is then displayed on the page each one. The user then can vote which ones are likely and which ones are unlikely. This is pretty much all done with Vue and not Vanilla javascript.

Phase 1: System Analysis

Data Used by the Program:

  • input from user : The user's IP address is obtained. The user can also click on different forecasts to vote on their likeliness.

Form the Output will Take:

The output will be a SPA built with Vue, with different sections for the top links, title, location, current weather, and predictions. It is interactable because the user can click and vote on the bottom forecasts.

Algorithms and Formula to be Used:

  • Vue Vue is used in the html instead of the lower-level template language (Django). Vue is also used in the javascript to create apps to mount to html elements to monitor their changes in the DOM. The data block contains the characteristics of the app.
  • fetch: There are 3 fetch calls. The second call is nested in the first call. All must be wrapped in a single try/catch block because the errors are pushed to the bottom anyway. They use promise chaining to avoid hadouken code.
  • methodsI'll probably need a method to deal with the dates and a method to toggle the color of the divs.
  • computed propertiesI'm going to use computed properties to monitor the number of likely, unlikely, and neutral votes in the page. I'm doing this because if one isn't changing but is being called multiple times, I want that information to be cached so it's not being recalculated.
